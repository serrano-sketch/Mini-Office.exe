# Mini-office

Mini-office es un editor de texto sencillo desarrollado en Python utilizando la biblioteca PySide6. Ofrece funcionalidades básicas de procesamiento de texto en una interfaz amigable.

## Características

- **Edición de Texto**: Escribir, copiar, cortar y pegar texto.
- **Gestión de Archivos**: Crear nuevos documentos, abrir archivos existentes y guardar cambios.
- **Búsqueda y Reemplazo**: Buscar texto dentro del documento y reemplazarlo (uno a uno o todo a la vez).
- **Personalización**: Cambiar el color de fondo del editor y la fuente del texto.
- **Contador de Palabras**: Visualización en tiempo real del número de palabras.
- **Autoguardado**: Guarda automáticamente los cambios cada 10 segundos si el archivo tiene nombre.
- **Interfaz**: Barra de herramientas, menús y barra de estado.

## Requisitos

- Python 3.x
- PySide6

## Instalación

1. Clona este repositorio o descarga el archivo `practicafinalalpha.py`.
2. Instala las dependencias necesarias:

```bash
pip install -r requirements.txt
```

## Uso

Para ejecutar la aplicación, corre el siguiente comando en tu terminal:

```bash
python practicafinalalpha.py
```
